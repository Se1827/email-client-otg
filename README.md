# Email Agent — OTG Client (Android + Wear OS)

Welcome to the **OTG (On-The-Go) Email Client**, a native Kotlin and Jetpack Compose Android application built for our hackathon submission! This repository contains the mobile and Wear OS frontend that pairs with our centralized FastAPI backend.

## Overview

The OTG client is designed to be a fast, secure, and lightweight frontend. Instead of running heavy LLMs on mobile devices (which drains battery and memory), the client delegates all intelligent processing—like email classification, PII masking, drafting, and calendar syncing—to the backend.

The result is a fluid, modern, and context-aware user experience on both your phone and watch.

## Key Features

- **Smart API Integration**: Clean decoupling of AI logic from the frontend. The app renders structured AI inferences (like Priority, Confidence Scores, and AI Drafts) generated by the backend.
- **Secure Local Cryptography**: Mirrors the backend's PBKDF2-HMAC-SHA256 authentication locally. Session tokens are securely stored using Android's `EncryptedSharedPreferences` (AES-256-GCM).
- **Contextual "Smart Cards"**: Uses Jetpack Compose to dynamically display emails based on their category (e.g., Finance, Flight, Meeting, Spam) for immediate context and safety.
- **AI-Powered Compose & Triage**: Frictionless UI to utilize AI tools for drafting and composing emails natively within the app.
- **Wear OS Extension**: Includes a standalone Wear OS module tailored for micro-interactions, letting you triage and approve AI-generated replies directly from your wrist.

## Project Structure

- `app/` — The main Android application (Jetpack Compose, Retrofit, OkHttp, Coroutines).
- `wear/` — The Wear OS companion app utilizing `SwipeDismissableNavHost`.

## Built With

- **Kotlin** & **Jetpack Compose**
- **Retrofit** & **OkHttp** for API communications
- AndroidX Security (EncryptedSharedPreferences)
- Jetpack Navigation (Mobile & Wear OS)
